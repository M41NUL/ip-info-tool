A Python CLI tool to get detailed information about any IP address.

## Installation
```bash
# Clone repository
git clone https://github.com/M41NUL/ip-info-tool.git

# Enter directory
cd ip-info-tool

# Install dependencies
pip install rich pyfiglet requests

# Run tool
python ip_tool.py
```

## Usage
```bash
# Option 1: Check your own IP
python ip_tool.py
# Then select 1

# Option 2: Check any IP
python ip_tool.py
# Then select 2 and enter IP
```

## Features
- Country, City, Region
- ISP & Organization
- Latitude/Longitude
- Google Maps Link
